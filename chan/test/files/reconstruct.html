It too supports floating mode, complete with fucking window controls, and has a root menu similar to openbox.

Openbox is a retarded, sort of crippled WM compared to i3 and awesome. Instead of being configured in a fairly plain json file or a lua clusterfuck, it's configured in an XML clusterfuck. So is its root menu. But you can like, shade the windows! They roll up into just a titlebar! It's so cute!

Clarification

Floating = windows style. Move boxes around by hand.

Manual tiling = tmux/screen style. i3 gives you four layouts and two splits. Splits are self explanatory and create new panes (the next window you open is inside the new pane), layouts are what you assign to the panes. The layouts are vertical stripes, horizontal stripes, tabs, and stacked titlebars. Herbstluftwm makes manual tiling more intuitive by creating an empty space where the new pane would be, but lacks a basic as fuck feature and has no tabs, stacks, or titles. i3 just makes the border adjacent to the area where the new split would be slightly brighter but actually floats transient windows instead of confusing your layout
